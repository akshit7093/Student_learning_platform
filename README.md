# Student Learning Platform

This is a comprehensive Student Learning Platform powered by AI.

## Features
- **Dashboard Overview**: View class statistics and student performance.
- **Student Reports**: Detailed AI-generated analysis of student profiles.
- **Job Analysis**: Analyze job descriptions and get tailored recommendations.
- **Resume Generation**: Create tailored resumes based on job roles.
- **Chat**: Interactive AI chat for student queries.

## Running Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app_copy.py`
